Proyecto a desarrollar

La aplicación contará con dos perfiles principales: Usuario Arrendatario y Usuario Arrendador. Usuario Arrendatario: Podrá crear, editar, eliminar y actualizar propiedades para arrendar, responder solicitudes de arrendamiento, aceptar o cancelar solicitudes, verificar pagos de arrendadores, y proporcionar calificaciones y comentarios. Usuario Arrendador: Podrá buscar propiedades para arrendar filtradas por ubicación, solicitar el arrendamiento de propiedades, realizar pagos y calificar tanto la finca arrendada como al arrendatario. Se integrará un sistema de autenticación y autorización para gestionar roles y permisos, y se desarrollarán interfaces de usuario intuitivas para una interacción eficiente. La planificación detallada y un enfoque centrado en el usuario son esenciales para el éxito de la plataforma.
